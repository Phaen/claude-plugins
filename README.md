# claude-plugins

Claude Code plugins by [Pablo Kebees](https://github.com/Phaen).

## Plugins

| Plugin | Description |
|---|---|
| [solve](solve/) | Structured problem-solving with an explicit solution tree and real-time visualisation. |
| [relay](relay/) | Session continuity — hand off context between sessions to split tasks, coordinate work, or resume later. |

## Install

Add this repo as a marketplace source in Claude Code, then install individual plugins:

```
/plugin marketplace add Phaen/claude-plugins
/plugin install solve@Phaen
/plugin install relay@Phaen
```
